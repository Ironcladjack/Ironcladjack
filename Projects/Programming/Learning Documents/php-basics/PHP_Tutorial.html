PHP Tutorial

Basic Layout

  • Structure:

    <body>
    <php

    // DO THE PHP HERE

    ?>
    </body>

  • Declaring Variables:

    <php

    $variable = "string" or value

    ?>

  • Commenting:

    <php

    // Comments Here

    ?>

  • Printing:

    <php

    echo "string" or value

    OR

    print "string" or value

    ?>

Modifiers

  • Length:

    $length = strlen("Jack");
    print $length

  • Substrings:

    $partial = substr("Jack", 0, 3);
    print $partial;

  • Uppercase:

    $uppercase = strtoupper($myname);
    print $uppercase;

  • Lowercase:

    $lowercase = strtolower(“Jack”);
    print $lowercase;

  • Search string for item:

    $variavle = strpos("Jack", "c");
    echo $variable

  • Rounding:

    $round = round(M_PI);
    print $round; // prints 3

    $round_decimal = round(M_PI, 4);
    print $round_decimal; // prints 3.1416

  • Random:

    // prints a number between 0 and 32767
    print rand();

    // prints a number between 1 and 10
    print rand(1,10);
