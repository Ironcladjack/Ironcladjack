Current Projects:

React.js Automation Server

This is a remake of the Node.js Automation Server i was working on before, however it has been remade using React.js, which I find more intuitive and easier to create distinct, repeatable components. This server controls: Philips Hue Lights, TP-Link Smart Plugs, several Arduino devices attached to: WS2812b RGB LED strips, Temperature/Humidity monitors, Stepper motors. It's currently hosted and visible on an AWS EC2 instance, however it is also currently running on a Raspberry Pi at my home.

Huebilight

This is the intended end-goal of my Node-Server project, to convert the web-app into a Desktop program, which will allow features like Ambilight, or Game Sense to be passed to Lights etc.

Currently can read an average screen colour and feed it to an RGB output, but I would like to finish working on the web-app before fully attempting to port it.

To Do App

This To Do app is a Node.js experiment to increase my knowledge of SQL/SQLite, as my experience with it prior to this was only used for single table applications.
It is built using Node.js and Angular, with several features planned, primarily multiple users with some form of authentication and notifications when a task is due.
It, like all my other viewable projects are hosted on an AWS EC2 instance., and attached it to my existing site as a subdomain.
