Control Phillips Hue using Node.js

Date Posted: 00/00/2010 Last Modified: 00/00/2010

In this tutorial, we'll walk through setting up and registering your server with a Phillips Hue Hub, then sending commands to change the status of any attached lights.

We'll use the node-hue-api package to do this.

An official walkthrough of controlling Hue via your browser can be found at: Phillips: Meet Hue

#Installing Package

The first thing we need to do is create a folder, initialise it, then install the node-hue-api package to Node.js. 

          // Create a folder and navigate to it 
          mkdir hue_control
          cd hue_control

          // initialise this folder and install packages 
          npm init
          npm install -g node-hue-api

#Registering a user

Now for the hub to accept any commands we would like to send, we need to register a valid user with the Hue Bridge.
The easiest way to do this is to follow the tutorial at Phillips: Getting Started
However, it would be more useful to write a script that searched for Hue Bridges on the local network, registered a new user with them and stored the resulting 'username' in a separate file.
